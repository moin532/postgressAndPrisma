
## API Reference

#### craete a employe

```http
  POST  http://localhost:300/api/users
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `\` | `string` | get users |

```http
  POST  http://localhost:300/api/users,
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `a` | `string` | create a employe |



```http
  GET http://localhost:300/api/users/:id
```
